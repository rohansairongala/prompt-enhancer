# ✨ AI Prompt Enhancer

A web app that transforms rough prompts into optimised versions 
tailored for specific AI models — powered by Claude API.

## What it does

Paste any prompt, select your target AI model, and get back 
three enhanced versions applying that model's published best practices.

**Supported models:** Claude · ChatGPT · Gemini · DeepSeek · Grok
